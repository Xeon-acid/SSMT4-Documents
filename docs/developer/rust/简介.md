# 简介
## Why Rust?
好用, 仅此而已.

## How Rust?
- 运行时速度优异;
- 内存安全;
- 多处理器适配良好.

## How Between Rust?
1. C/C++ 性能出色, 但类型系统与内存不安全;
2. Java/C# 使用 Garbage Collector, 内存安全, 诸多特性优秀, 然性能捉襟见肘;
3. Rust:
    - 安全;
    - 易于维护调试;
    - 高效.

## Where to Rust?
1. 高性能 Web Service;
2. WASM (Web Assembly);
3. 命令行工具;
4. 网络编程;
5. 嵌入式;
6. **系统编程**.

## Why not Rust?
- 难学.

## How to Rsut?
Rust 相比目前大多主流语言存在诸多独特概念, 需要一步一步学, 否则会懵.
