# 项目说明
SSMT4-Documents项目基于VitePress2.0系列开发

# 安装vitepress

`bun add -D vitepress@next`

# 初始化项目
`bun vitepress init`

# 运行测试

`bun run docs:dev`


# 开始写文档

第一次克隆后，需要进入项目目录中初始化vitepress环境
```
bun install
```

后续即可随时本地测试
```
bun run docs:dev
```
